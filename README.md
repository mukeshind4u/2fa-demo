# 🔐 2FA Demo (PHP Authentication System)

A simple and secure login system with Two-Factor Authentication (2FA) using OTP.

---

## 🚀 Features

* User Registration & Login
* OTP Verification (2FA)
* Secure Password Hashing
* Session Management
* QR Code Support (if added)

---

## 🛠️ Tech Stack

* PHP (Core)
* MySQL
* HTML / CSS

---

## ⚙️ Installation

1. Clone the repository:
   git clone https://github.com/mukeshind4u/2fa-demo.git

2. Move to project folder:
   cd 2fa-demo

3. Setup database:

* Create database: `2fa_demo`
* Import `auth_test.sql`

4. Update database config:
   config/database.php

5. Run project:
   http://localhost/2fa-login/login.php
